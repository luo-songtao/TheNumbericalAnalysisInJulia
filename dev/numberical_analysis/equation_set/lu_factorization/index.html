LU分解–高斯消去法的矩阵形式

EquationSet.lu_factorization — Method

LU分解–高斯消去法的矩阵形式

lu_factorization(A, b, n)

将系数矩阵分解为下三角矩阵L和上三角矩阵U，再求解x，可降低运算次数。

LU分解使得在朴素的高斯消元法中的消去过程，向量b没有参与运算，直到回代过程才参与运算(两次)，相比而言，当n非常大时，计算次数会有明显的差异

注意：当前实现的算法如果遇到0主元会抛出异常并终止

LU分解–高斯消去法的矩阵形式推导：

$\begin{aligned} A &= LU \\ Ax &=b \Longleftrightarrow LUx=b \\ Lc &= b \Rightarrow c \\ Ux &=c \Rightarrow x \end{aligned}$

Arguments

  • A: 表示系数矩阵A
  • b: 表示常数项b
  • n: 方程数

Example

julia> A = Float64[1 2 -1;2 1 -2;-3 1 1]
3×3 Array{Float64,2}:
  1.0  2.0  -1.0
  2.0  1.0  -2.0
 -3.0  1.0   1.0
julia> b = Float64[3; 3; -6]
3-element Array{Float64,1}:
  3.0
  3.0
 -6.0
julia> x = lu_factorization(A,b,3)
3×1 Array{Float64,2}:
 3.0
 1.0
 2.0

PA=LU分解–部分主元法

EquationSet.lu_factorization_with_permutation — Method

PA=LU分解

lu_factorization_with_permutation(A, b, n)

为避免淹没问题，在对矩阵A进行LU分解前，先对主元列进行判断，将主元列最大的一行与当前的首行进行置换

PA=LU分解推导：

$\begin{aligned} P &: 置换矩阵 \\ A = LU \Longleftrightarrow PA&=LU \end{aligned}$

Arguments

  • A: 表示系数矩阵A
  • b: 表示常数项b
  • n: 方程数

Example

julia> A = Float64[2 1 5; 4 4 -4; 1 3 1]
3×3 Array{Float64,2}:
 2.0  1.0   5.0
 4.0  4.0  -4.0
 1.0  3.0   1.0
julia> b = Float64[5; 0; 6]
3-element Array{Float64,1}:
 5.0
 0.0
 6.0
julia> x = lu_factorization_with_permutation(A,b,3)
3×1 Array{Float64,2}:
 -1.0
  2.0
  1.0
